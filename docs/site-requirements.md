# BWNA WordPress Site Requirements

## Site Information

**Current Site**: https://www.bwnapdx.org/
**Platform**: Wix
**Target Platform**: WordPress (hosted on WordPress.com)
**Organization**: Beaumont-Wilshire Neighborhood Association (Portland, OR)

---

## Content Requirements

### Pages
- Complete migration of all existing pages
- Maintain page hierarchy and structure
- Preserve all text content, images, and media

### Blog
- Migrate existing blog posts
- Preserve post dates, authors (if applicable)
- Maintain any categories or tags

### Forms
- **Two Mailchimp signup forms** (embedded iframes)
- Forms must function identically to current site
- Preserve form styling and placement

---

## Technical Requirements

### Must Have
- Responsive design (mobile, tablet, desktop)
- Fast page load times
- SEO-friendly URLs and metadata
- Mailchimp integration working
- All images optimized for web

### Nice to Have
- Contact form (if not already present)
- Social media integration
- Google Analytics integration
- Newsletter archive (if blog posts exist)

---

## Design Requirements

### Brand Consistency
- Match current color scheme exactly
- Use same or similar typography
- Maintain visual hierarchy
- Preserve logo and branding elements

### Layout
- Replicate current page layouts
- Maintain navigation structure
- Preserve footer content and links

---

## Hosting

**Target Platform**: WordPress.com hosting
- Theme must be compatible with WordPress.com
- Should work with WordPress.com's plugin restrictions
- Consider using WordPress.com's Mailchimp block or embed options

---

## Success Criteria

✅ All pages migrated with identical content
✅ Design closely matches current site
✅ Mailchimp forms working correctly
✅ Blog posts migrated and functional
✅ Site is mobile responsive
✅ All images and media transferred
✅ Navigation menus replicated
✅ SEO metadata preserved
