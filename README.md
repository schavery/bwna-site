# Wix to WordPress Migration Project

This repository contains all the tools, scripts, and documentation for migrating a Wix website to WordPress.

## Quick Start

1. Review the [Migration Plan](MIGRATION_PLAN.md) for the complete project roadmap
2. Provide your Wix site URL and requirements
3. Run the scraping tools to extract content
4. Develop the custom WordPress theme
5. Import content and deploy

## Repository Structure

- `MIGRATION_PLAN.md` - Comprehensive migration plan and checklist
- `docs/` - Documentation including site inventory and style guide
- `scrapers/` - Web scraping scripts for content extraction
- `scraped-content/` - Extracted content and media assets
- `wordpress/wp-content/themes/custom-theme/` - Custom WordPress theme
- `scripts/` - Utility scripts for import and setup
